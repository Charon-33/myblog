## 3. 使用 @vue/cli 搭建项目

- 方法一：[使用@vue/cli](https://cli.vuejs.org/zh/)

（1）安装脚手架

```shell
npm install -g @vue/cli
# OR
yarn global add @vue/cli
```

查看版本信息

```shell
vue --version
```

升级脚手架

```shell
npm update -g @vue/cli
# 或者
yarn global upgrade --latest @vue/cli
```

（2）创建项目

```shell
vue create [project-name]
```
